Apache nifi Downloads

Apache NiFi Project Keys can be used to verify downloads.

Please allow up to 24 hours for mirrors to synchronize following the release of a new version.

Previous releases can be found in release archives. Please avoid repeated downloads from archives to avoid infrastructure rate limits.
