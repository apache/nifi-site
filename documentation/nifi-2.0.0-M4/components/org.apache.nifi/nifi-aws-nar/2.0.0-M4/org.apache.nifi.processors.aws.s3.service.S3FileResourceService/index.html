S3FileResourceService

Description:

State management:

Restricted:

Example Use Cases:

Use Case:

Fetch a specific file from S3. The service provides higher performance compared to fetch processors when the data should be moved between different storages without any transformation.

Configuration:

"Bucket" = "${s3.bucket}"

"Object Key" = "${filename}"

The "Region" property must be set to denote the S3 region that the Bucket resides in.

The "AWS Credentials Provider Service" property should specify an instance of the AWSCredentialsProviderService in order to provide credentials for accessing the bucket.
