PublishGCPubSubLite

Description:

Publishes the content of the incoming FlowFile to the configured Google Cloud PubSub Lite topic. The processor supports dynamic properties. If any dynamic properties are present, they will be sent along with the message in the form of 'attributes'.

Tags:

google, google-cloud, gcp, message, pubsub, publish, lite
