Apache nifi Mailing Lists

These are the mailing lists that have been established for this project. For each list, there is a subscribe, unsubscribe, and an archive link. Tips on how to interact with ASF mailing lists found here

Name Subscribe Unsubscribe Post Pony Mail Archive Legacy Archive Purpose
Users Mailing List Subscribe Unsubscribe Post users@nifi.apache.org mail-archives.apache.org For nifi users to get help, ask questions, report bugs
Developers Mailing List Subscribe Unsubscribe Post dev@nifi.apache.org mail-archives.apache.org For developers to discuss features, bugs, share ideas
Commits Mailing List Subscribe Unsubscribe Post commits@nifi.apache.org mail-archives.apache.org Automated e-mails each time code commits occur
Issues Mailing List Subscribe Unsubscribe Post issues@nifi.apache.org mail-archives.apache.org Automated e-mails each time comments are made in JIRA or Github
Security Mailing List private private Post not archived not archived Alias to report security issues to NiFi. Please see ASF security reporting guidance and NiFi security page
