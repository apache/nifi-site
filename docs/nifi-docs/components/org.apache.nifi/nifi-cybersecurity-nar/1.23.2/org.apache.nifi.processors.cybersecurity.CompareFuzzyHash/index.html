CompareFuzzyHash

Description:

Compares an attribute containing a Fuzzy Hash against a file containing a list of fuzzy hashes, appending an attribute to the FlowFile in case of a successful match.

Tags:

hashing, fuzzy-hashing, cyber-security

Properties:
