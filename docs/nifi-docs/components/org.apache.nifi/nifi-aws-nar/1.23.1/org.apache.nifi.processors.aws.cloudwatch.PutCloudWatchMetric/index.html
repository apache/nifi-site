PutCloudWatchMetric

Description:

Publishes metrics to Amazon CloudWatch. Metric can be either a single value, or a StatisticSet comprised of minimum, maximum, sum and sample count.

Tags:

amazon, aws, cloudwatch, metrics, put, publish
