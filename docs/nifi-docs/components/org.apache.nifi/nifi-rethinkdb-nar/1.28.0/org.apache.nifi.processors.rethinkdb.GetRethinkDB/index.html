GetRethinkDB

Deprecation notice:

RethinkDB 2.4 requires API changes that would alter the configuration and behavior of this Processor

Please consider using one the following alternatives: No alternative components suggested.

Description:

Processor to get a JSON document from RethinkDB (https://www.rethinkdb.com/) using the document id. The FlowFile will contain the retrieved document

Tags:

rethinkdb, get, read, fetch
